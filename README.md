玩法一览
--------------------------------

提出更好的玩法  


配套条件
--------------------------------
vs2015（推荐社区版）  
  
windows7以上（推荐win8.1）  

C#学习书籍：  
zapline在用《C＃高级编程（第九版）》  
<http://item.jd.com/11611762.html>  
网上可以找到英文完整版  

其它工具：  
MarkdownPad（.md）  
Visio 2013（.vsdx）  
Mockplus（.mp）  


编码风格
--------------------------------
4个空格缩进  
大括号单独占一行  
其它注意保持统一，开心就好
